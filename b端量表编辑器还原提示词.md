# B端量表编辑器还原提示词规范

请 1:1 还原截图中的 B 端量表编辑器，技术栈可使用纯 HTML/CSS/JS 或 React + Ant Design 5.x。

## 页面结构
- 顶部固定栏：左侧「退出」+ 量表名称「测试」，右侧按钮「预览 / 保存 / 发布」。
- 主体四栏：左侧窄导航 54px + 题型面板 254px + 中央题目画布 + 右侧属性面板 260px。
- 中央画布背景为浅灰点阵，题目区域为白色长画布，选中题目显示蓝色描边。

## 题型面板
- 分类：选择、文本输入、高级题型、矩阵。
- 选择：单选、多选。
- 文本输入：单行文本。
- 高级题型：评分打分、日期。
- 矩阵：矩阵单选、矩阵多选、矩阵打分、矩阵题。
- 点击任一题型后，在中央画布末尾新增对应题目，并自动选中。

## 矩阵题新增要求
- 新增题型名称为「矩阵题」。
- 默认展示为矩阵填空：行标题 + 多个列，每个单元格为输入框，初始所有列的数据类型均为填空。

## 右侧属性面板
- 右侧面板顶部是唯一的 tab 区域：默认只显示「题目设置」，选中具体选项后在「题目设置」后面追加「选项设置」，不要在内容区重复渲染第二组 tab。
- 通用字段：题目类型下拉、是否必填开关。
- 单选/多选：最多可选、最少可选、批量编辑选项。
- 点击单选/多选题中的具体选项后，右侧属性面板需要出现「题目设置 / 选项设置」tab，并默认切到「选项设置」。
- 点击矩阵类题目的具体列头时，也需要把该列当作一个选项，右侧出现「题目设置 / 选项设置」tab，并默认切到「选项设置」。
- 选项设置中「当前选项」放在第一位。
- 指标绑定使用一个支持下拉候选和输入检索的文本框，候选为数据库字段。
- 普通单选/多选的选项设置只展示：当前选项、指标绑定；不展示字段类型。
- 字段类型只在矩阵类题目的列选项设置中展示，选项包含：下拉选择、单选、多选、填空、评分、日期。
- 矩阵列字段类型为下拉选择、单选、多选时，展示对应值域配置。
- 矩阵列字段类型为填空时，展示文本类型配置。
- 矩阵列字段类型为评分时，展示评分模板和评分样式配置。
- 矩阵列字段类型为日期时，展示日期格式配置。
- 单行文本：文本格式、最多填写、提示文案。
- 评分打分：文案模板、显示样式。
- 日期：日期格式。
- 矩阵类题型：矩阵单选、矩阵多选、矩阵打分、矩阵题都需要在「批量编辑行」上面展示「允许患者新增行」开关。
- 矩阵题：属性结构先与矩阵单选/矩阵多选保持一致，包含每行最多可选、每行最少可选、允许患者新增行、批量编辑行、批量编辑列。

## 交互要求
- 点击题目卡片选中，右侧属性面板同步切换。
- 修改右侧属性，中央题目实时更新。
- 选中题目显示「复制 / 删除」浮动工具条。
- 至少保留一道题，不能删空。
